## 📌 Deskripsi Aplikasi
  Aplikasi ini merupakan sistem manajemen data aktivitas lari yang memungkinkan pengguna untuk mencatat dan mengelola riwayat olahraga mereka. Dengan aplikasi ini, pengguna dapat menyimpan informasi seperti tanggal lari, jarak tempuh, durasi, serta data pendukung lainnya secara terstruktur. Aplikasi ini bertujuan untuk membantu pengguna dalam memantau perkembangan aktivitas lari secara mudah dan efisien.

## ⚙️ Fitur Utama
1. Tambah Data Lari (Create)
    Pengguna dapat menambahkan data aktivitas lari ke dalam sistem, seperti tanggal, jarak, dan durasi.
2. Tampilkan Data Lari (Read)
    Sistem menampilkan daftar seluruh data lari yang telah tersimpan sehingga pengguna dapat melihat riwayat aktivitasnya.
3. Edit / Update Data Lari (Update)
    Pengguna dapat memperbarui data lari jika terdapat kesalahan atau perubahan informasi.
4. Hapus Data Lari (Delete)
    Pengguna dapat menghapus data lari yang sudah tidak diperlukan dari sistem.

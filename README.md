# Aplikasi Catat Lari
## 📌 Deskripsi Proyek
Aplikasi ini merupakan sistem sederhana untuk mengelola data aktivitas lari. Pengguna dapat mencatat, melihat, mengubah, dan menghapus data lari dengan mudah. Sistem ini dirancang untuk membantu pengguna dalam memonitor aktivitas lari mereka secara terstruktur dan efisien.

## ⚙️ Fitur Utama
1. Tambah Data Lari (Create)
Pengguna dapat menambahkan data aktivitas lari seperti tanggal, jarak tempuh, durasi, dan informasi pendukung lainnya.
2. Tampilkan Data Lari (Read)
Sistem dapat menampilkan seluruh data lari yang telah tersimpan sehingga pengguna dapat melihat riwayat aktivitas mereka.
3. Edit / Update Data Lari (Update)
Pengguna dapat mengubah data lari yang sudah ada apabila terdapat kesalahan atau ingin memperbarui informasi.
4. Hapus Data Lari (Delete)
Pengguna dapat menghapus data lari yang tidak diperlukan dari sistem.
